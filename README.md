Коллекция проектов, выполненных в рамках изучения FPGA и языков описания аппаратуры:

 - arrow_neuron - Проект по разработке модели нейроускорителя для решения задачи распознавания стрелки
 - command_processing_system - Cистема обработки команд, состоящая из трех функциональных блоков: анализатора команд, обработчика команд, генератора сообщений.
 - sea_battle - Проект реализации графического интерфейса классической игры «Морской бой» на плате FPGA.
 - sequence_analyzer - Проект анализатора последовательности согласно варианту с последующим выводом истории правильно выведенных значений на семисегментный динамический индикаторт и пользовательские светодиоды рабочей платы.
 - sequence_generator - Проект генератора последовательности согласно варианту с последующим выводом значение на светодиоды и на семисегментный динамический индикатор.
 - serial_hex_calculator - Проект цифрового устройства, реализующего одну арифметическую операцию над парой чисел в шестнадцатеричной системе счисления, взаимодействующее с персональной ЭВМ (ПЭВМ) по последовательному интерфейсу UART.
