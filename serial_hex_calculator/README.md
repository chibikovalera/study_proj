Проект цифрового устройства, реализующего одну арифметическую операцию над парой чисел в шестнадцатеричной системе счисления, взаимодействующее с персональной ЭВМ (ПЭВМ) по последовательному интерфейсу UART.

В качестве обработчика информации, поступающей от ПЭВМ, выступает функциональный блок, содержащий конечный автомат, который реализуетанализ принятых данных, выполняет над принятыми данными арифметическую
операцию (согласно варианту) и формирует ответное сообщение, отправляемоена ПЭВМ по интерфейсу UART. Арифметическая операция выполняется наддвумя операндами: над полученным в принятом сообщении числом и надсодержимым внутреннего регистра конечного автомата. Также конечныйавтомат должен обеспечивать вывод на ПЭВМ по интерфейсу UART тестовых
сообщений в кодировке ASCII, содержащих информацию об ошибках (в случаеих возникновения).
